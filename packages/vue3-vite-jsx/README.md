## vue3-vite-jsx

基于 vue3+vite3+tsx+tailwind 开发前端基础项目框架


## 快速开始

### 获取项目

#### 使用 Git

```sh

git clone git@github.com:geallenboy/garron-vue-template.git

```

### 安装依赖

```sh
pnpm install

```

### 启动项目

```sh
pnpm run start
```

### 项目打包

```sh
pnpm run build
```

## Git 贡献提交规范

- 参考 Angular 规范
  - feat 增加新功能
  - fix 修复问题/BUG
  - style 代码风格相关无影响运行结果的
  - perf 优化/性能提升
  - refactor 重构
  - revert 撤销修改
  - test 测试相关
  - docs 文档/注释
  - chore 依赖更新/脚手架配置修改等
  - workflow 工作流改进
  - ci 持续集成
  - types 类型定义文件更改
  - wip 开发中
